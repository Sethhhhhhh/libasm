# Libasm
### A simple assembly library 
###### Status : on Progress



## Installation

Install [NASM](http://macappstore.org/nasm/) and [GCC](https://discussions.apple.com/thread/8336714) on your MacOS. Only compatible under a 64-bit MacOS system.

```shell
make && gcc libasm.a main.c
```
